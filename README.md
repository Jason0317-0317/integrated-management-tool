# 春不老薪資計算系統

整合管理工具 — 結合教練堂數統計、業績報表轉換、出勤明細統計和業務獎金計算的完整解決方案。

## 技術棧

- **前端/後端**: Python Streamlit（單一應用，無需分離部署）
- **資料處理**: Pandas, NumPy, OpenPyXL

## 專案結構

```
integrated-management-tool/
├── app.py              # Streamlit 主應用
├── requirements.txt    # Python 依賴
├── .env                # 環境變數（本地開發用，不提交 Git）
└── README.md           # 本文件
```

## 安裝與運行

### 1. 安裝依賴

```bash
pip install -r requirements.txt
```

### 2. 設定環境變數

建立 `.env` 檔案（或在伺服器上直接設定）：

```env
FINANCE_PASSWORD=your-strong-password-here
```

> **重要**：請勿將密碼寫死在程式碼中，也不要將 `.env` 提交到 Git。

### 3. 啟動應用

```bash
streamlit run app.py
```

應用會在 `http://localhost:8501` 啟動。

## 功能說明

### 店長用

#### 1. 教練堂數統計
- 支援多館別篩選（全部、中山館、高美館、義昌館、巨蛋館）
- 靈活的日期區間選擇
- 自動名稱轉換（20+ 位老師的簡稱對應）
- 支援 Excel 和 CSV 格式
- 自動課程分類統計
- 生成橫向統計表，支援下載 Excel

#### 2. 團個績計算工具
- 自動計算單價、折數、未稅金額
- 智能分類新購和續購
- 自動計算業績獎金（2%、1.5%、1%）
- 生成分類統計表，支援下載 Excel

#### 3. 出勤明細統計
- 上傳進出場記錄
- 自動統計出勤天數與總工時
- 產生出勤明細、月份統計、月曆排班表三個工作表

### 小編用

#### 業務獎金計算系統

| 獎金類型 | 說明 | 計算方式 |
|---------|------|--------|

## 故障排除

**Q: 財務登入顯示「系統尚未設定財務密碼」**
- 確認已設定環境變數 `FINANCE_PASSWORD`
- 本地開發請建立 `.env` 檔案並安裝 `python-dotenv`
- 雲端部署請在平台的環境變數設定中加入此變數

**Q: 檔案上傳失敗**
- 確認檔案格式正確（Excel 或 CSV）
- 確認檔案包含必要欄位
- 查看頁面上的錯誤訊息

**Q: 獎金計算不正確**
- 確認員工身份設定（正職/兼職）影響品牌知名度基準人數
- 驗證各項輸入數據是否正確
